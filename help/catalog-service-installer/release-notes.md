# [!DNL Catalog Service Installer] Release Notes

These release notes describe updates to the **[!DNL Catalog Service Installer]** (`magento/catalog-service-installer`)—the Composer metapackage that registers the Catalog Service feature for your SaaS environment. For what the installer does and how it relates to the main Catalog Service extension, see the [[!DNL Catalog Service Installer] overview](overview.md).

Support is provided for the **latest** released version. Release notes for earlier versions are kept for reference.

Updates are categorized by type:

![New](../assets/new.svg) New features
![Fix](../assets/fix.svg) Fixes and improvements
![Bug](../assets/bug.svg) Known issues

>[!NOTE]
>
> Storefront Catalog Service also publishes release notes for the **Catalog Service API and metapackage** in the [[!DNL Catalog Service] release notes](../catalog-service/release-notes.md). Check both guides when you upgrade PHP, Adobe Commerce, or Composer.
